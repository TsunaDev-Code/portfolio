# Инструкция по настройке отправки писем

## Требуемые шаги:

### 1. Установка пакета Resend

```bash
npm install resend
```

### 2. Получение API ключа

1. Перейдите на https://resend.com
2. Зарегистрируйтесь или войдите в аккаунт
3. В разделе API Keys создайте новый ключ
4. Скопируйте ключ

### 3. Настройка переменных окружения

Добавьте в файл `.env.local` (создайте его, если его нет):

```env
RESEND_API_KEY=your_resend_api_key_here
```

### 4. Проверка домена (опционально)

Если вы хотите отправлять письма с вашего собственного домена, нужно добавить DNS записи в Resend.

## Текущая конфигурация:

- **От кого отправляются письма**: `noreply@tsuna-dev.com`
- **На кого приходят письма**: `info@tsuna-dev.com`
- **Ответ идет на**: Email адрес посетителя (для Reply-To)

Если нужно изменить эти адреса, отредактируйте файл:
`src/app/api/send-email/route.ts`

## Альтернативные способы отправки писем:

Если вы хотите использовать другой сервис (SendGrid, Gmail SMTP и т.д.), вы можете отредактировать файл `src/app/api/send-email/route.ts` и заменить логику отправки письма на нужный вам сервис.
